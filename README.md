# AgriNode 🌾 
### Empowering Farmers with Data-Driven Market Intelligence

AgriNode is a lightweight, mobile-first web application designed to bridge the information gap for farmers in India. By providing instant access to verified Mandi (market) prices, AgriNode enables farmers to make informed decisions about when and where to sell their produce.

[Live Demo](https://agriii-node.vercel.app/) 

## 🚀 The Mission
In some regions, farmers often lack real-time price transparency. AgriNode aggregates fragmented government data into a single, user-friendly dashboard, putting the latest market trends right at their fingertips.

## 🛠️ Tech Stack
- **Frontend:** React + Vite
- **Language:** TypeScript (Type-safe data fetching)
- **Styling:** Tailwind CSS (Mobile-responsive UI)
- **API:** RESTful Integration with Open Government Data (OGD) Platform India
- **Deployment:** Vercel (CI/CD Pipeline)

## ✨ Key Features
- **Mandi Price Tracking:** View Min, Max, and Modal prices for various commodities.
- **Advanced Filtering:** Filter data by State, District, Date, and Commodity.
- **User-Defined Date Logic:** Purposefully built to handle API reporting lags, allowing users to verify historical and recent data.
- **Optimized for Low Connectivity:** A lightweight build ensures the app remains functional even on limited 3G/4G networks.

## 📈 What I Learned
- **Asynchronous Data Handling:** Managing complex API states and error boundaries in React.
- **TypeScript Interfaces:** Defining strict types for agricultural datasets to ensure app stability.
- **Product Thinking:** Prioritizing MVP features based on the actual needs of the end-user (farmers).
